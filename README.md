# Sales & Revenue Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze sales and revenue performance across products, categories, regions, and time periods.

## Features

* KPI Cards for Total Revenue, Total Units Sold, and Average Order Value
* Monthly Revenue Trend Analysis
* Top Products by Revenue
* Revenue Distribution by Category
* Interactive Filters and Slicers
* Business Insights Summary

## Tools Used

* Power BI
* Microsoft Excel
* DAX Measures

## Dataset

A simulated sales dataset containing:

* Product information
* Categories
* Regions
* Units Sold
* Unit Prices
* Revenue

## Key Insights

* Laptop Pro generated the highest revenue.
* Smartphone X was the second best-selling product.
* Electronics contributed over 96% of total revenue.
* Revenue peaked during January.
* Product revenue is concentrated in a small number of high-value products.
